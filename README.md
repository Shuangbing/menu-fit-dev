## Smart Order System

![menu-fit-logo](./docs/logo.png)

## Usage
### Install
- frontend -> client & admin -> yarn / npm install
- server -> yarn / npm install

### Develop Server
- frontend -> client & admin -> yarn dev / npm run dev
- server -> nest start

### Build
- frontend -> client & admin -> yarn build / npm run build
- server -> nest build
